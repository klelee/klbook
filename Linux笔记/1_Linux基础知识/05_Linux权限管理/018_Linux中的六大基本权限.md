### rwx权限详解

#### rwx作用到文件

1. 【r】代表可读
2. 【w】代表可写：可以修改，但不代表可以删除文件，删除一个文件的前提条件是对该文件所在的目录有可写权限
3. 【x】代表可执行

#### rwx作用到目录

1. 【r】代表可读，可以使用ls
2. 【w】代表可写：可以修改，可以创建，修改，删除文件
3. 【x】代表可执行：可以进入该目录

#### 用数字表示rwx

```bash
r = 4
w = 2
x = 1
```

### 改变文件权限

修改一个文件对应的权限使用的命令是chmod，这个命令有两种修改权限的方式，一种是通过数字的形式修改，另一种是通过符号的形式修改。

#### 通过数字的形式修改文件的权限

```bash
chmod 755 abc    # 将abc文件的权限修改为755，也就是，rwxr-xr-x
chmod 755 -R abcd/    # 递归修改abcd文件夹下所有文件的权限值为755
```

#### 通过符号的形式修改文件的权限

```bash
chmod u=rwx,go=rx abc    # 同样是将abc文件的权限修改为755，也可以这样使用符号赋值来修改。
```
