## 权限的基本介绍

### ls -l中显示内容

![image-20211214164438451](https://klelee-image.oss-cn-qingdao.aliyuncs.com/image/image-20211214164438451.png)

第一列表示各用户对当前文件/目录的权限，0-9位说明：

- 第0位（上图黄色）：表示文件类型
  - \- 表示普通文件
  - d 表示目录
  - c 表示字符文件，如，鼠标，键盘
  - b 表示块设备
  - l 表示链接
- 第1-3位（上图绿色）：表示文件/目录所有者的权限  ---user
- 第4-6位（上图蓝色）：表示文件/目录所在组的权限  ---group
- 第7-9位（上图红色）：表示其他组权限   ---other

### rwx权限详解

#### rwx作用到文件

1. 【r】代表可读
2. 【w】代表可写：可以修改，但不代表可以删除文件，删除一个文件的前提条件是对该文件所在的目录有可写权限
3. 【x】代表可执行

#### rwx作用到目录

1. 【r】代表可读，可以使用ls
2. 【w】代表可写：可以修改，可以创建，修改，删除文件
3. 【x】代表可执行：可以进入该目录

#### 用数字表示rwx

```
r = 4; w = 2; x = 1.    因此，rwx=4 + 2 + 1
```


## 修改权限--chmod

### 第一种方式：+、-、=变更权限

u：所有者

g：所有组

o：其他人

a：所有人

1. `chmod  u=rwx,g=rx,o=x 文件/目录名`
2. `chmod o+w 文件/目录名`
3. `chmod a-x 文件/目录名`

### 第二种方式：通过数字变更权限

```
r = 4; w = 2; x = 1.    因此，rwx=4 + 2 + 1
```

#### 使用方式

`chmod <number> <filename>` 举例 `chmod 751 hello.py`
